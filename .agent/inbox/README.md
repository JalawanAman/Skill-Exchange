# Inbox

This is where Jalawan drops things for the agent to pick up.

**Drop anything here:**
- Screenshots of UI inspiration
- Notes or voice-to-text dumps
- New requirements or ideas
- Reference links saved as `.txt` files
- Images, PDFs, anything relevant

**Agent reads this at the start of every session.**  
Once something has been processed, the agent moves it to `context/` or `decisions/` if it shaped a decision, or deletes it if it was a one-time note.
